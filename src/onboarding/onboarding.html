CoverMe uses Google's Gemini AI to create your personalized cover letters

You'll need a free Google Gemini API key to use this extension. Here's why:

  • Your key ensures your data stays private
  • You control your own API usage
  • Google offers generous free tier access

How to get your API key:

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Create a new API key (it's free)
  4. Copy and paste it below

Don't have a key right now? You can set it up later in the extension settings.
